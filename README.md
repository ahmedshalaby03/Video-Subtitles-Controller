# 🎬 Interactive Video Subtitles Controller | Vanilla JS

مشروع متقدم للتحكم في واجهة مشغل الفيديو والترجمات المصاحبة (Subtitles) برمجياً، مما يتيح تجربة مستخدم تفاعلية ومرنة. يعتمد المشروع على معالجة مسارات النصوص (Text Tracks) برمجياً لضمان تحكم كامل في كيفية عرض الترجمة وتوقيتها.

---

### ✨ المميزات الرئيسية
* **Multi-Language Support**: دعم كامل للتبديل السلس بين الترجمة العربية والإنجليزية للمقطع المرئي بضغطة زر.
* **Subtitles Toggle**: إمكانية إخفاء الترجمة تماماً (Turn Off) أو إعادة إظهارها فوراً باستخدام منطق الـ **JavaScript**.
* **Custom Control Bar**: واجهة تحكم مخصصة مصممة برمجياً لضمان سهولة الوصول لخيارات اللغة والترجمة أسفل المشغل.
* **Responsive Video Layout**: مشغل فيديو متجاوب تماماً مع مختلف مقاسات الشاشات لضمان أفضل تجربة مشاهدة للمستخدم على كافة الأجهزة.

---

### 🛠 التكنولوجيات المستخدمة

| التقنية | الاستخدام |
| :--- | :--- |
| **HTML5 Video API** | استيراد وإدارة ملف الفيديو والمسارات النصية (Subtitles Tracks) بشكل برمي. |
| **JavaScript (Vanilla)** | برمجة منطق التبديل بين اللغات (Active Tracks) والتحكم في عناصر الـ UI وتفاعلها. |
| **CSS3** | تنسيق الأزرار (Buttons) وتصميم واجهة المشغل بشكل عصري وجذاب يتماشى مع معايير الـ UI الحديثة. |
