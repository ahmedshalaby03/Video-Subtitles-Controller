Interactive Video Subtitles Controller 🎬
مشروع متقدم للتحكم في واجهة مشغل الفيديو والترجمات المصاحبة (Subtitles) برمجياً، مما يتيح تجربة مستخدم تفاعلية ومرنة.
✨ المميزات الرئيسية:
Multi-Language Support: دعم كامل للتبديل بين الترجمة العربية والإنجليزية للمقطع المرئي.

Subtitles Toggle: إمكانية إخفاء الترجمة تماماً (Turn Off) أو إظهارها بضغطة زر باستخدام الـ JavaScript.

Custom Control Bar: واجهة تحكم مخصصة تضمن سهولة الوصول لخيارات اللغة والترجمة.

Responsive Video Layout: مشغل فيديو متوافق مع مختلف مقاسات الشاشات لضمان أفضل تجربة مشاهدة.
🛠 التكنولوجيات المستخدمة:
HTML5 Video API: لاستيراد وإدارة ملف الفيديو والمسارات الصوتية والنصية.

JavaScript (Vanilla): لبرمجة منطق التبديل بين اللغات (Active Tracks) والتحكم في عناصر الـ UI.

CSS3: لتنسيق الأزرار (Buttons) وتصميم واجهة المشغل بشكل عصري وجذاب.
